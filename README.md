# dm5.user.js

DM5 动漫屋滚动阅读，一次展示所有图片，点击图片切换至下一页。

SPDX-License-Identifier: GPL-3.0-or-later

This is a userscript initiated from [@violentmonkey/generator-userscript](https://github.com/violentmonkey/generator-userscript).

## Development

``` sh
# Compile and watch
$ pnpm run dev

# Build script
$ pnpm run build

# Lint
$ pnpm run lint
```
